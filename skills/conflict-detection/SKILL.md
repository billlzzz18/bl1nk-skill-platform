# Conflict Detection

## Description

Use this skill to identify the core conflicts driving a story, evaluate whether stakes escalate properly, and reveal places where tension is unclear or too weak.

## Workflow

1. Extract the major sources of conflict.
2. Separate internal, relational, and external pressures.
3. Evaluate escalation, consequences, and resolution paths.
4. Recommend changes that sharpen stakes and urgency.
