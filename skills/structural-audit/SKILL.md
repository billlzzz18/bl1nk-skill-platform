# Structural Audit

## Description

Use this skill to review story structure, identify missing plot beats, flag pacing problems, and recommend concrete revisions that improve narrative coherence.

## Workflow

1. Identify the current story structure or intended framework.
2. Map key beats, turning points, and escalation.
3. Highlight missing setup, payoff, or transition issues.
4. Recommend prioritized revisions with rationale.
