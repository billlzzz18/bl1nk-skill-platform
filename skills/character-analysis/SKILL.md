# Character Analysis

## Description

Use this skill to inspect protagonists, antagonists, and supporting characters for clarity, motivation, growth, contradiction, and role effectiveness.

## Workflow

1. List named and implied characters.
2. Summarize each character's goal, need, conflict, and arc.
3. Evaluate consistency of actions, voice, and relationships.
4. Suggest focused improvements that strengthen the cast.
