# Arc Optimization

## Description

Use this skill to improve narrative progression by tightening setup/payoff links, clarifying turning points, and strengthening emotional or thematic momentum.

## Workflow

1. Outline the current arc shape.
2. Identify weak transitions, reversals, or underdeveloped payoffs.
3. Propose revisions that improve progression and cohesion.
4. Present changes in a practical rewrite order.
