# Project Goals

## Vision
To create the definitive desktop-first IDE for AI development, empowering developers to build, test, and manage complex Claude AI skills and agents with enterprise-grade security and developer ergonomics.

## Core Objectives

### 1. Unified AI Development Interface
Provide a single, powerful environment that replaces fragmented tools for prompt engineering, skill management, and model testing.

### 2. Enterprise-Grade Security
Ensure all API credentials and sensitive data are stored locally and encrypted using industry-standard AES-256-GCM, giving users full control over their secrets.

### 3. Multi-Provider Flexibility
Support a wide array of LLM providers including AWS Bedrock, Anthropic, OpenAI, Google, OpenRouter, and local models (Ollama, LM Studio) to ensure developers are never locked into a single ecosystem.

### 4. Developer Productivity
Include a pre-built library of 501+ expert-crafted agents and 15+ skill templates to accelerate the start of any AI project.

### 5. Robust Quality Assurance
Provide integrated testing tools, including automated validation for skills and agents, and a suite of 80+ E2E tests to ensure the platform's reliability.

## Principles

- **Human-in-the-loop**: All AI actions are mediated and approved by the user.
- **Privacy First**: Local-first architecture ensures user data stays on their machine unless explicitly shared.
- **Type-Safe Everything**: Leverage TypeScript 5.8 and tRPC for end-to-end type safety across the entire stack.
- **Open Standards**: Support for Model Context Protocol (MCP) and OpenAPI 3.1 specifications.
